# Telegram alerts in muted channels on a specific-text
This code will send you alerts by telegram bot in any your channel if new post/message or any was found with specific text that you want

Starting
1. Download python (i am used 3.13)
2. Download required libraries (pip install telethon / asyncio / qrcode / aiogram
3. go to main.py by any txt editor
4. get api id and api hash from [my telegram](https://my.telegram.org/)
5. set yours text in username
6. get bot and his token from bot father in telegram
7. create channel for notifications, add bot in it, set bot to admin
8. set channel @id in notify_chat
9. start programm
10. join by qr
